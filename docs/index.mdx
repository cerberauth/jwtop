## Quick start

```sh
# Install
go install github.com/cerberauth/jwtop@latest

# Decode a token
jwtop decode $TOKEN

# Verify with an HMAC secret
jwtop verify $TOKEN --secret mysecret

# Probe a server for JWT vulnerabilities
jwtop crack $TOKEN --url https://api.example.com/protected
```
