# JWTop

**JWTop** is a JWT operations toolkit for Go. It ships as both a standalone CLI and an importable Go library, covering the full token lifecycle: decoding, verification, creation, re-signing, and security testing.

## What you can do

| Task | CLI | Library |
|---|---|---|
| Inspect a token without a key | `jwtop decode` | `jwt.Decode()` |
| Verify a signature | `jwtop verify` | `jwt.Verify()` |
| Create a signed token | `jwtop create` | `jwt.Create()` |
| Re-sign with a different key or algorithm | `jwtop sign` | `editor.TokenEditor` |
| Probe a server for JWT vulnerabilities | `jwtop crack` | `crack.ProbeAll()` |
| Produce an exploited token for testing | `jwtop exploit` | `exploit.*` |
| Brute-force an HMAC secret | `jwtop exploit weaksecret` | `exploit.CrackSecret()` |

## Quick start

```sh
# Install the CLI
go install github.com/cerberauth/jwtop@latest

# Decode a token
jwtop decode eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJ1c2VyMTIzIn0.abc

# Verify with an HMAC secret
jwtop verify $TOKEN --secret mysecret

# Create a signed token
jwtop create --alg HS256 --secret mysecret --sub user123 --exp 1h

# Probe a server for JWT vulnerabilities
jwtop crack $TOKEN --url https://api.example.com/protected
```
