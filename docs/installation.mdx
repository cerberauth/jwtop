## CLI

<Tabs>
  <TabItem label="go install">

```sh
go install github.com/cerberauth/jwtop@latest
```

Requires Go 1.24 or later.

  </TabItem>
  <TabItem label="Homebrew">

```sh
brew install cerberauth/tap/jwtop
```

  </TabItem>
  <TabItem label="Docker">

```sh
docker run --rm ghcr.io/cerberauth/jwtop <command>
```

  </TabItem>
  <TabItem label="From source">

<Steps>

1. Clone the repository:

   ```sh
   git clone https://github.com/cerberauth/jwtop.git
   cd jwtop
   ```

2. Build the binary:

   ```sh
   go build -o jwtop .
   ```

3. Move it to your `PATH`:

   ```sh
   mv jwtop /usr/local/bin/
   ```

</Steps>

  </TabItem>
</Tabs>

Verify the installation:

```sh
jwtop version
```

## GitHub Actions

To use `jwtop` in a GitHub Actions workflow, see the [GitHub Actions guide](./github-actions/).

<Aside type="tip">
  The [`cerberauth/jwtop-action`](https://github.com/cerberauth/jwtop-action) composite action installs and runs `jwtop` with no extra setup.
</Aside>
